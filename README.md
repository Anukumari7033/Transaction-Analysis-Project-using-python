# Transaction-Analysis-Project-using-python
This project focuses on analyzing transactions data using Python's Pandas library .It demonstrates how to perform data cleaning ,exploration, and visualization to gain insights from transactional datasets.
Features :- 
. Data cleaning: Handling missing values, duplicates, and formatting issues.
.Exploratory Data Analysis (EDA): Summarizing the dataset, calculating statistics abd undersstanding patterns.
.Filtering & Quering : Efficiently filters transactions based on date,amount,or other criteria
.Visualization: create histogram,bar chart to visualize transactions distributions and trends
. Insights : Identify high - value transactions, frequent transactions dates and other trends.

TECH STACK:
1. python3.x
2. Pandas - Data manipulation and analysis
3. Matplotlib/Seaborn - Data Visualization


Purpose : This project is ideal foe beginners in data analysis to practice real - world data handling and gain insight using 
          Python and Pandas.
